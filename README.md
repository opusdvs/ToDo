# Todo (Frontend)

Это любительский проект. Я только начал осваивать web-разработку, поэтому код здесь служит в первую очередь для практики и экспериментов.

Технологии:
- HTML
- CSS
- JavaScript

Как запустить:
- Откройте `index.html` в браузере
- Или запустите локальный сервер (в терминале проекта):
  - python3 -m http.server 8000
  - открыть http://localhost:8000

Примечание:
Код может содержать ошибки и не предназначен для продакшна.
